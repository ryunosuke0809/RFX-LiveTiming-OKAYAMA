# Live Timing 位置情報の許可手順（一般向け案内）

**対象 URL:** https://mola-timing-okayama.com/  
**対象:** 岡山国際サーキット場内で Live Timing を閲覧するお客様

---

## はじめに（なぜ位置情報が必要か）

Live Timing（一般向け）は、**岡山国際サーキット場内のお客様向け**のサービスです。  
場外からの閲覧を防ぐため、ブラウザの**位置情報**を使って場内にいることを確認しています。

- 初回アクセス時に「位置情報を許可しますか？」と表示されたら **「許可」** を選んでください
- 一度「許可しない」を選ぶと、下の手順で設定を開き直す必要があります
- 場外へ移動した場合は、自動的に表示が止まります

> 関係者向けの別 URL をお使いの方は、この手順は不要です。

---

## うまくいかないときのチェック（先に確認）

| 症状 | 確認すること |
|------|----------------|
| 「許可してください」と出る | 下の端末別手順で、サイトの位置情報を **許可** にする |
| 「取得できませんでした」 | 端末本体の位置情報サービスを **オン** にする／屋外や窓際で再試行 |
| 「会場の範囲外」 | サーキット場内にいるか確認。場外では閲覧できません |
| ずっとくるくる回る | 約10秒待っても変わらなければ「再試行」。位置情報オフのままのことがあります |

**共通のコツ**

1. 端末の **位置情報（Location）自体がオン** になっていること  
2. ブラウザで **このサイトだけ許可** になっていること（本体がオンでもサイト拒否だと使えません）  
3. 設定変更後はページを **再読み込み** する  

---

## iPhone（Safari）

### A. 初回のポップアップが出たとき

1. Safari で https://mola-timing-okayama.com/ を開く  
2. 「〜があなたの位置情報を使用しようとしています」→ **許可**

### B. 以前「許可しない」を選んだ場合

1. **設定** アプリを開く  
2. **プライバシーとセキュリティ** → **位置情報サービス**  
3. 位置情報サービスが **オン** か確認  
4. 下にスクロールして **Safari のウェブサイト**（または **Safari**）を開く  
5. **許可** または **この App の使用中** を選ぶ  
6. Safari に戻り、ページを再読み込み  

### C. サイト単位で拒否している場合

1. **設定** → **アプリ** → **Safari**（機種により「設定 → Safari」）  
2. **位置情報** または **Webサイトの設定** 関連を確認  
3. または Safari でサイトを開いた状態で、アドレス欄左の **ぁあ** / **ページ設定** → **位置情報** → **許可**  
4. 再読み込み  

---

## iPhone（Chrome）

1. **設定** → **プライバシーとセキュリティ** → **位置情報サービス** を **オン**  
2. 同じ画面で **Chrome** を開き、**Appの使用中** または **許可**  
3. Chrome で https://mola-timing-okayama.com/ を開く  
4. 許可ダイアログが出たら **許可**  
5. 拒否済みの場合: アドレス欄左の錠／チューリップアイコン → **権限** → **位置情報** → **許可** → 再読み込み  

---

## iPad（Safari / Chrome）

手順は **iPhone と同じ** です。

- 本体: **設定 → プライバシーとセキュリティ → 位置情報サービス**  
- Safari / Chrome それぞれに位置情報の許可が必要です  
- サイトを開いて拒否した場合は、アドレス欄のサイト設定から **位置情報 → 許可**  

---

## Mac（Safari）

1. **システム設定** → **プライバシーとセキュリティ** → **位置情報サービス**  
2. 位置情報サービスを **オン**  
3. 一覧の **Safari**（または Safari のウェブサイト）にチェックを入れる  
4. **Safari** → **設定** → **Webサイト** → **位置情報**  
5. `mola-timing-okayama.com` を **許可**（「拒否」になっていたら変更）  
6. ページを再読み込み（**⌘⇧R** 推奨）  

> Mac には GPS が無いため、Wi‑Fi などで位置を推定します。屋内では取得に失敗することがあります。その場合は「再試行」を押すか、窓際・別ブラウザでお試しください。

---

## Mac（Chrome）

1. **システム設定** → **プライバシーとセキュリティ** → **位置情報サービス** を **オン**  
2. 一覧の **Google Chrome** にチェック  
3. Chrome でサイトを開く → 許可ダイアログで **許可**  
4. 拒否済みの場合: アドレス欄左のアイコン → **位置情報** → **許可**  
5. または Chrome **設定** → **プライバシーとセキュリティ** → **サイトの設定** → **位置情報** で、このサイトがブロックされていないか確認  
6. 再読み込み（**⌘⇧R**）  

---

## Windows（Microsoft Edge）

1. Windows の **設定** → **プライバシーとセキュリティ** → **位置情報**  
2. **位置情報サービス** を **オン**  
3. 必要に応じて **デスクトップアプリが位置情報にアクセスすることを許可する** もオン  
4. Edge で https://mola-timing-okayama.com/ を開く → **許可**  
5. 拒否済みの場合: アドレス欄左の錠アイコン → **サイトのアクセス許可** → **位置情報** → **許可**  
6. または Edge **設定** → **Cookie とサイトのアクセス許可** → **位置情報** で確認  
7. ページを再読み込み（**Ctrl + F5**）  

---

## Windows（Google Chrome）

1. Windows の **位置情報** を **オン**（上記 Edge と同じ）  
2. Chrome でサイトを開く → **許可**  
3. 拒否済みの場合: アドレス欄左のアイコン → **位置情報** → **許可**  
4. または Chrome **設定** → **プライバシーとセキュリティ** → **サイトの設定** → **位置情報**  
5. 再読み込み（**Ctrl + Shift + R**）  

---

## Windows（Safari）

Windows 向け Safari は現在ほぼ使われていません。  
**Edge** または **Chrome** のご利用をおすすめします。

## 場内でのおすすめの使い方（掲示・案内用・短文）

```
【Live Timing のご利用について】
URL: https://mola-timing-okayama.com/

・場内でのみご覧いただけます
・表示されたら「位置情報」を許可してください
・許可しないと画面が開きません
・うまくいかないときは端末の位置情報をオンにし、
  ブラウザでこのサイトの位置情報を「許可」にして再読み込みしてください
```

## スタッフ向けメモ（お客様対応）

| 質問 | 回答の目安 |
|------|------------|
| なぜ位置情報が必要？ | 場内の方だけが閲覧できるようにするためです |
| 個人情報は？ | 閲覧可否の判定に使い、地図アプリのように常時追跡する用途ではありません |
| 家でも見たい | 一般向け URL は場内限定です。場外ではご覧いただけません |
| iPhone は見れるが PC はダメ | PC は GPS が無く測位が不安定なことがあります。スマホのご利用もご案内ください |
| 許可したのに場外と出る | 場内にいるか、ブラウザを再読み込み。それでもダメなら別ブラウザで試す |
